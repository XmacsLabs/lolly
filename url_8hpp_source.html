104/******************************************************************************
105 * url routines by label
106 * + "" (empty string): ., .., ..., /, /tmp
107 * + none: invalid url
108 * + root: the url http://gnu.org yields (concat (root "http") "gnu.org");
109 * + concat: a/b/c is represented as (concat "a" (concat "b" "c"));
110 * + or: the path a:b/c is represented as (or "a" (concat "b" "c"));
111 * + wildcard: (wildcard) corresponds to any url
