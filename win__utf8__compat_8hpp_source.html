1/******************************************************************************
2 * MODULE : win-utf8-compat.hpp
3 * DESCRIPTION: The windows version of TeXmacs makes use of
4 * Boost.Nowide standalone library for accessing
5 * the Windows UTF-16 file API
6 *******************************************************************************
7 * As most application originating from the *nix world, TeXmacs
8 * uses UTF-8 encoding for files names, file pathes and
9 * environment variables. Porting such applications to Windows
10 * is complicated due to the fact that Windows API only accepts
11 * UTF-16 for unicode.
12 *
13 * Boost.Nowide provides an elegant light weight solution to this problem,
14 * making porting from *nix to windows UTF-16 unicode API mostly transparent
15 *
16 * See http://cppcms.com/files/nowide/html/index.html
17 *
18 * Texmacs specific changes to the original Boost.Nowide standalone files dated
19 * 2017-Jun-10 are indicated in nowide/texmacs_modifications.patch
20 * ******************************************************************************
21 * LICENCE :
22 * Boost.Nowide is released under Boost Software License - Version 1.0.
23 * see https://www.boost.org/users/license.html
24 ******************************************************************************/
